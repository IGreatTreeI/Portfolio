  • HTML

    "인터넷의 가치는 html의 전과 후로 나뉜다."
    html은 가장 기초적인 웹 프로그래밍 언어였습니다.
    오랜 전통과 유지력을 보여준 html을 먼저 배워봤습니다.
    가장 기초적인 html 기본 양식과 태그들에 대해서 배우니 얼마나 혁신적인 기술인지 알게 되었습니다.
    더 나아가 body, a, li, div, br, p 여러 가지 태그들도 배웠습니다.
    서툴지만 배운 것을 바탕으로 코드를 써 내려가며 웹페이지를 만들었습니다.
    지금 보여드릴 것은 제가 html을 이용해 만든 첫 웹사이트입니다. (클릭)

  • CSS

    "CSS는 html에 폭발적인 디자인 효과를 만들어 냈다."
    처음 만든 페이지는 상당히 투박하고 단조로웠습니다.
    그래서 웹페이지를 꾸미고 싶다는 여러 가지 불만족들이 생겨났습니다.
    CSS가 그런 저의 불만족들을 채워주었습니다.
    이제는 font로 글자 색을 바꾸고 border로 선을 그릴 수 있게 되었습니다.
    class나 id로 개체를 선택하는 법도 알게 되었습니다.
    CSS 한 줄로 제 웹사이트는 폭발적으로 변신해 나갔습니다.
    이번에 보여드릴 것은 첫 웹사이트를 기초적인 CSS만을 이용해 꾸민 것입니다. (클릭)

  • JavaScript

    "정보가 살아움직이고 상호작용하는 것은 마법 같은 일이다."
    이제 어느 정도 웹페이지라 부를 수 있게 되었습니다.
    그러나 아직 만족스럽지 못했습니다.
    제가 만든 페이지는 단순히 정적인 사이트에 지나지 않았습니다.
    이젠 사이트에 동적인 효과를 부여 넣고 싶어졌습니다.
    그때 만난 게 javascript였습니다.
    function을 배우고 조건문 반복문 같은 문법을 알게 되었습니다.
    html이나 CSS보다 훨씬 난이도가 올라갔지만, 이젠 진짜 코딩을 하는 것처럼 느껴졌습니다.
    마지막으로 보여드릴 것은 야간 모드 버튼을 추가한 웹페이지입니다. (클릭)
